# AgentHub 1.0.0 — first stable release

The first stable release, and the biggest update yet: a brand-new **Companion Tools** section
for the CLI utilities that work *alongside* your coding agents, plus seven new agents.

<!-- more -->

## Companion Tools

Companion tools are **not coding agents** but CLI utilities that operate on the same ecosystem —
usage/cost tracking, context packing, skill management. They get their own section in the toolbar
dropdown and in Settings, reuse the same install / detect / update / launch flow, and are
**off by default** (opt-in). When launched they show a 🧩 marker in the terminal tab to set them
apart from agents (🤖).

- **[ccusage](https://ccusage.com)** — token & cost usage reports from local agent logs.
- **[Repomix](https://repomix.com)** — pack your entire repository into a single AI-friendly file.
- **[Skills](https://www.skills.sh)** — install reusable agent skills across many CLI agents.
- **[TokenTracker](https://www.tokentracker.cc)** — local-first token & cost dashboard for 25 AI tools.

Enable the ones you want from **Settings/Preferences → Tools → AgentHub → Companion Tools**.

## Seven new agents

- **[CodeBuddy](https://www.codebuddy.ai)** — Tencent's AI-powered terminal coding agent.
- **[Command Code](https://commandcode.ai)** — a CLI coding agent that learns your coding taste.
- **[Devin](https://devin.ai/cli)** — Cognition's local agent with Devin Cloud handoff.
- **[iFlow CLI](https://iflow.cn)** — terminal intelligence for repos and workflow automation.
- **[Junie CLI](https://junie.jetbrains.com)** — JetBrains' LLM-agnostic coding agent.
- **[Kode](https://github.com/shareAI-lab/Kode-cli)** — an open-source, terminal-native agent.
- **[OpenClaw](https://openclaw.ai)** — an open-source, Claude Code–style assistant. 🦞

That brings AgentHub to **38 built-in agents**.

> **Note:** Command Code is hidden on Windows — its launch command `cmd` collides with the
> built-in Windows `cmd.exe`.

## Get it

Install from the [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/32310-agenthub)
or search for **AgentHub** in **Settings/Preferences > Plugins > Marketplace**.
