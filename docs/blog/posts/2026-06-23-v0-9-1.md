# AgentHub 0.9.1 — Freebuff agent & wider IDE support

A small follow-up to the first public release: a new agent, wider IDE support, and a
fix to the toolbar install flow.

<!-- more -->

## What's new

- **New agent — [Freebuff](https://freebuff.com/cli)** by Codebuff. The free,
  ad-supported coding agent for your terminal.
- **Wider IDE support** — the minimum platform is now **JetBrains IDE 2024.1+
  (build 241+)**, down from 2025.1, so AgentHub installs on more IDE versions.

## Fixes

- **Toolbar install honors the background setting** — installing a not-installed
  agent from the toolbar dropdown now follows the *"Run operations in background"*
  setting. When an agent is installed in the
  background, it now launches automatically once it's ready.

## Get it

Install from the [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/32310-agenthub)
or search for **AgentHub** in **Settings/Preferences > Plugins > Marketplace**.
