# AgentHub 0.9.0 — First public release

This is the first public release of **AgentHub**, a JetBrains IDE plugin that
launches any CLI coding agent directly in its own IDE terminal window — one click
from the toolbar.

<!-- more -->

## What's in it

- **30 built-in agents** — Aider, Amp, Antigravity CLI, Auggie, Claude Code, Cline,
  Codex CLI, Cody CLI, Continue CLI, Copilot CLI, Crush, Cursor CLI, Droid, ForgeCode,
  Goose CLI, Grok Build, Kilo Code, Kimi Code, Kiro CLI, LeanCTL, MiMo Code, Mistral
  Vibe, OpenCode, OpenHands, Plandex, Qodo, Qoder CLI, Qwen Code, SWE-agent and VT Code.
- **Auto-detection** — AgentHub scans your `PATH` on first install and after each
  update, marking which agents are actually installed. Results are persisted across
  IDE restarts, with the last run timestamp shown in Settings.
- **Install flow** — agents that aren't installed are labeled *(not installed)*
  and offer to install via a confirmation dialog; success is confirmed automatically
  in the background.
- **Update notifications** — a background check on IDE startup notifies you when
  npm/pip-based agents have newer versions available.
- **Check & Update utilities** — run version checks or updates across all installed
  agents at once.
- **Custom agent support** — add your own CLI tool with a name, command, and URL.
- **Cross-platform** — works on macOS, Linux, and Windows.

## Requirements

- JetBrains IDE 2025.1+ (platform build 251+)
- The bundled Terminal plugin

## Get it

Install from the [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/32310-agenthub)
or search for **AgentHub** in **Settings/Preferences > Plugins > Marketplace**.
