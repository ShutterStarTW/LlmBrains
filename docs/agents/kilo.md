# Kilo Code - by Kilo

## Installation

```shell
# Install with npm
npm install -g @kilocode/cli

# Update to latest version
npm update --quiet --no-fund -g @kilocode/cli

# Uninstall
npm uninstall -g @kilocode/cli
```

> via [kilo.ai](https://kilo.ai)


## Get Version

```
% kilo --version
```

## Usage

Kilo Code is an open-source AI coding agent forked from Cline, offering autonomous coding capabilities with full file system and terminal access.

```
Usage: kilo [options] [prompt]

Kilo Code - Open-source autonomous coding agent CLI

Options:
  -v, --version      Output the version number
  -h, --help         Display help for command
```

## Features

- Open-source, community-driven fork of Cline
- Full file system access (read, write, create, delete)
- Terminal command execution with approval flow
- Supports all major LLM providers
- Plan & Act modes for complex tasks
