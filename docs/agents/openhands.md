# OpenHands - by All Hands AI

## Installation

```shell
# Install with pip
pip install openhands-ai

# Update to latest version
pip install --upgrade --upgrade-strategy eager openhands-ai

# Uninstall
pip uninstall -y openhands-ai
```

> via [openhands.dev](https://openhands.dev/)


## Get Version

```
% openhands --version
```

## Usage

OpenHands is a platform for AI-powered software development agents. The CLI provides terminal access to OpenHands agents that can browse the web, write and execute code, manage files, and call APIs.

```
Usage: openhands [options] [command]

OpenHands - AI software development agent CLI

Options:
  -v, --version      Output the version number
  -h, --help         Display help for command
```

## Features

- Web browsing and research capabilities
- Code execution in a sandboxed environment
- Full file system management
- API integration and automation
- Supports multiple LLM backends (OpenAI, Anthropic, etc.)
- Docker-based sandbox for safe code execution
