# Freebuff - by Codebuff

## Installation

```shell
# Install with npm
npm install -g freebuff

# Update to latest version
npm update --quiet --no-fund -g freebuff

# Uninstall
npm uninstall -g freebuff
```

> via [freebuff.com/cli](https://freebuff.com/cli)


## Get Version

```
% freebuff --version
```

## Usage

Freebuff is the free, ad-supported coding agent that runs in your terminal — no API key,
no credit card, and no subscription. Point it at any codebase, new or existing, and
describe what you want done.

```
% cd your-project
% freebuff
# describe a task in natural language and let the agent edit your codebase
```

## Features

- Free to use, supported by text ads instead of subscription fees
- Works on macOS, Windows, and Linux (requires Node.js 18+)
- No API key or credit card required to get started
- Runs against any new or existing codebase from the terminal
