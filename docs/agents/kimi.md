# Kimi Code - by Moonshot AI

## Installation

```shell
# Install with pip
pip install kimi-cli

# Update to latest version
pip install --upgrade --upgrade-strategy eager kimi-cli

# Uninstall
pip uninstall -y kimi-cli
```

> via [kimi.com/code](https://www.kimi.com/code)


## Get Version

```
% kimi --version
```

## Usage

Kimi Code is Moonshot AI's CLI coding agent powered by the Kimi k1.5 and k2 models, optimised for long-context code tasks.

```
Usage: kimi [options] [prompt]

Kimi Code - Moonshot AI coding agent CLI

Options:
  -v, --version      Output the version number
  -h, --help         Display help for command
```

## Features

- Long context window (up to 128k tokens) for large codebases
- Powered by Kimi k1.5 / k2 reasoning models
- Chinese and English language support
- Code generation, debugging, and refactoring
