# Continue CLI - by Continue

## Installation

```shell
# Install with npm
npm install -g @continuedev/cli

# Update to latest version
npm update --quiet --no-fund -g @continuedev/cli

# Uninstall
npm uninstall -g @continuedev/cli
```

> via [continue.dev](https://continue.dev)


## Get Version

```
% cn --version
```

## Usage

Continue CLI (`cn`) brings the Continue coding assistant to your terminal, enabling AI-powered development outside the IDE extension.

```
Usage: cn [options] [prompt]

Continue CLI - AI coding assistant in your terminal

Options:
  -v, --version      Output the version number
  -h, --help         Display help for command
```

## Features

- Context-aware assistance using your local codebase
- Supports 100+ LLMs via Continue's model configuration
- Shares configuration with the Continue IDE extension
- Customisable with `.continue/` project config
