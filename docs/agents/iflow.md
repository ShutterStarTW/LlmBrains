# iFlow CLI - by iFlow

## Installation

```shell
# Install with npm
npm install -g @iflow-ai/iflow-cli

# Update to latest version
npm update --quiet --no-fund -g @iflow-ai/iflow-cli

# Uninstall
npm uninstall -g @iflow-ai/iflow-cli
```

> via [platform.iflow.cn/en/cli/quickstart](https://platform.iflow.cn/en/cli/quickstart) &middot; [github.com/iflow-ai/iflow-cli](https://github.com/iflow-ai/iflow-cli)

## Get Version

```
% iflow --version
```

## Usage

iFlow CLI is a command-line intelligence that embeds in your terminal, analyzes your repositories,
performs coding tasks, and automates workflows. Authenticate with iFlow's native login (free) or
an OpenAI-compatible API.

```
% cd your-project
% iflow
# describe a task in natural language and let the agent work
```

## Features

- Repository analysis and coding tasks from the terminal
- Native iFlow auth (free) or OpenAI-compatible API
- Workflow automation from simple file ops to complex pipelines
