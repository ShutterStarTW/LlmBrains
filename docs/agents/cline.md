# Cline - by Cline

## Installation

```shell
# Install with npm
npm install -g cline

# Update to latest version
npm update --quiet --no-fund -g cline

# Uninstall
npm uninstall -g cline
```

> via [cline.bot/cli](https://cline.bot/cli)


## Get Version

```
% cline --version
```

## Usage

Cline is an autonomous coding agent that can read/write files, run terminal commands, use a browser, and create/manage MCP servers — all with your approval.

```
Usage: cline [options] [prompt]

Cline - Autonomous coding agent CLI

Options:
  -v, --version      Output the version number
  -h, --help         Display help for command
```

## Features

- Full file system access (read, write, create, delete)
- Terminal command execution with approval flow
- Browser automation for testing and research
- MCP server management
- Supports OpenAI, Anthropic, Google, and local models
- Plan & Act modes for complex multi-step tasks
