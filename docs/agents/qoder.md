# Qoder CLI - by Qoder AI

## Installation

```shell
# Install with npm
npm install -g @qoder-ai/qodercli

# Update to latest version
npm update --quiet --no-fund -g @qoder-ai/qodercli

# Uninstall
npm uninstall -g @qoder-ai/qodercli
```

> via [qoder.com](https://qoder.com)


## Get Version

```
% qodercli --version
```

## Usage

Qoder CLI is an AI-powered coding assistant that helps you write, review, and refactor code directly from your terminal.

```
Usage: qodercli [options] [prompt]

Qoder CLI - AI coding assistant

Options:
  -v, --version      Output the version number
  -h, --help         Display help for command
```

## Features

- Intelligent code generation and completion
- Code review and quality suggestions
- Refactoring assistance
- Multi-language support
