# CodeBuddy - by Tencent

## Installation

```shell
# Install with npm (requires Node.js 22+)
npm install -g @tencent-ai/codebuddy-code

# Update to latest version
npm update --quiet --no-fund -g @tencent-ai/codebuddy-code

# Uninstall
npm uninstall -g @tencent-ai/codebuddy-code
```

> via [codebuddy.ai/docs/cli/installation](https://www.codebuddy.ai/docs/cli/installation)

## Get Version

```
% codebuddy --version
```

## Usage

CodeBuddy Code is Tencent's AI-powered CLI coding agent. It brings large-model capabilities
directly into your terminal with deep codebase awareness, editing files and running commands
in your environment.

```
% cd your-project
% codebuddy
# describe a task in natural language and let the agent work
```

## Features

- Terminal-native coding agent with full codebase context
- Edits files and executes commands in your environment
- Works on macOS, Linux, and Windows (requires Node.js 22+)
