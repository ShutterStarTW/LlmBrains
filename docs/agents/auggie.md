# Auggie - by Augment Code

## Installation

```shell
# Install with npm
npm install -g @augmentcode/auggie

# Update to latest version
npm update --quiet --no-fund -g @augmentcode/auggie

# Uninstall
npm uninstall -g @augmentcode/auggie
```

> via [augmentcode.com/product/CLI](https://www.augmentcode.com/product/CLI)


## Get Version

```
% auggie --version
```

## Usage

Auggie is the CLI coding agent from Augment Code. It brings the power of Augment's deep code understanding to your terminal.

```
Usage: auggie [options] [prompt]

Auggie - Augment Code CLI agent

Options:
  -v, --version      Output the version number
  -h, --help         Display help for command
```

## Features

- Deep codebase understanding powered by Augment's indexing
- Persistent memory across sessions
- Multi-file edits with full project context
- Works with your existing IDE setup
